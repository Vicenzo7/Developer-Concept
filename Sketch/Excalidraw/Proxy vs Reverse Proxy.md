Forward Proxy
-> No client can talk directly to the server. They have to talk to a proxy server

Advantage
1. In Schools and Institution, to restrict certain content
2. To hide ones identity -> (hides IP address)
3. Groups Similar request and send it as one request because it is same.
4. Brings Security.
5. Can also access restricted content because request is coming from proxy to the internet.
6. Caching -> Contents are stored which are fetched from internet.

Disadvantage
1. Forward Proxy is at application layer, means for each application we have to set up a proxy. ^x6ik0Ago

Reverse Proxy
    -> Any request coming from the internet is not allowed to talk directly to the server.

Advantage
1. Security, Internet does not know the  IP of your internal servers. So in case of DDOS attack, attacker can 
attack only the reverse proxy.
eg: CDN is reverse PROXY.

2. Caching
3. Latency
4. Load Balancer ^s1zM8cJL
